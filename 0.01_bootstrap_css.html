Adding Bootstrap into you html

You can add the bootstrap to any html doc with lang="en" inside the opening html tag.

Mobile First

To ensure proper rendering and touch zooming add the following:

If you add maximum-scale=1, user-scalable=no, this prevents the user from zooming

Containers

Using .container will fix the width responsively. .container-fluid will span the entire width of your viewport.

Columns

The whole page is set up in columns of 12.

You can nest columns and get them to wrap for smaller screens.

Media Queries

@media is how to set min and max screen widths

Typography

The typography can be manipulated with BS also.

You can change alignment, and transform from upper to lower case.
